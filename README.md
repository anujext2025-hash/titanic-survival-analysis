# Titanic Survival Analysis 🚢

## Overview
Predicting Titanic passenger survival using Random Forest.

## Results
- Accuracy: 81%
- ROC-AUC: 0.89

## Key Findings
- Females survived at 74% vs 18% for males
- First class (62.9%) vs third class (24.2%) survival
- Children had highest survival rate at 57.9%

## Tech Stack
Python, Pandas, Seaborn, Scikit-learn
